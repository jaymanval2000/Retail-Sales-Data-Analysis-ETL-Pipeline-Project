# Retail-Sales-Data-Analysis-ETL-Pipeline-Project
End-to-end Retail Sales ETL project using SQL, Python, and Power BI to transform raw sales data into actionable business insights through interactive dashboards.

Project Overview:

The objective of this project is to transform raw retail sales data into meaningful insights by:

Extracting structured data using SQL

Cleaning and transforming data using Python

Visualizing key metrics through an interactive Power BI dashboard

This project simulates a real-world data analytics workflow commonly used in MIS and Data Analyst roles.
